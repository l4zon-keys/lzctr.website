LazCouture— Private Access Gateway
This repository contains the source code for the exclusive, access-only gateway page for LazCouture. The page serves as a secure storefront and pre-launch landing page designed to restrict access to authorized users, brand insiders, and early community members via a secure password or access-key entry.

🚀 Features
Minimalist UI/UX: A sleek, high-end visual aesthetic aligned with the brand's identity.

Gatekeeper Access: Secured entry field preventing unauthorized access to the main platform.

Responsive Design: Optimized for seamless performance across mobile, tablet, and desktop devices.

Pre-Launch Mailing List: Integrated data capture to build a community before the official drop.
